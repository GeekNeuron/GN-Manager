GN Manager

A powerful and professional suite of command-line tools to automate software management on Windows.

About The Project

GN Manager is a comprehensive solution for Windows power users. Instead of a single, monolithic script, it's a suite of four specialized, high-performance tools that work in harmony. The suite includes the GN Winget Install Manager for modern online installations, the GN Local Install Manager for rapid offline setups, the GN Cleaner Manager for deep system cleaning, and the GN Backup Manager for vital data and settings protection. Each tool is designed to be powerful on its own, but together they form the ultimate toolkit for Windows automation.

The Tool Suite

Focuses on modern, online software management using the official Windows Package Manager (Winget). It allows you to search, install, and upgrade thousands of applications directly from their official repositories, ensuring you always have the latest versions.

A simple yet powerful drag-and-drop utility designed for offline or bulk installations. Just drop a folder containing your `.exe` and `.msi` installers onto the script, and it will attempt to install all of them silently and sequentially.

Your all-in-one uninstallation and system cleanup tool. It intelligently lists installed programs, uninstalls them, then hunts down leftover files (moving them to a safe quarantine) and orphaned registry keys (deleting them with your explicit, per-key confirmation).

The ultimate safety net for power users. Define profiles for complex applications (like IDEs or creative software) in the config file. This tool can then export all their critical data, settings, and registry keys for easy restoration later, saving countless hours of reconfiguration.
