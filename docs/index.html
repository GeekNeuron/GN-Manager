GN Manager

A complete and professional suite of command-line tools designed to automate and master software management on Windows.

About The Project

GN Manager is a comprehensive solution for Windows power users. Instead of a single, monolithic script, it's a suite of four specialized, high-performance tools that work in harmony. The suite includes the GN Winget Install Manager for modern online installations, the GN Local Install Manager for rapid offline setups, the GN Cleaner Manager for deep system cleaning, and the GN Backup Manager for vital data and settings protection. Each tool is designed to be powerful on its own, but together they form the ultimate toolkit for Windows automation.

Explore the full project and its source code on the official GitHub repository.

The Tool Suite

This tool is your gateway to the modern Windows Package Manager ecosystem. A simple yet powerful drag-and-drop utility designed for offline or bulk installations. Just drop a folder containing your `.exe` and `.msi` installers onto the script, and it will attempt to install all of them silently and sequentially.

It goes beyond the standard uninstaller by deep-scanning for leftover files (which it safely moves to a quarantine folder) and orphaned registry keys (deleting them with your explicit, per-key confirmation).
