GN Manager

A complete and professional suite of command-line tools designed to automate and master software management on Windows.

About The Project

The GN Manager project was born out of the need for a powerful, centralized, and intelligent solution for Windows power users. Instead of relying on a single, monolithic script, the suite is architected as a collection of four specialized, high-performance tools that work in harmony. This modular design ensures that each tool is an expert in its domain, providing a clean, focused, and robust user experience.

From modern online package management to deep system cleaning and vital data protection, GN Manager provides a comprehensive toolkit for developers, system administrators, and anyone who wants to take full control of their Windows environment. Explore the full project and its source code on the official GitHub repository.

The Tool Suite

This tool is your gateway to the modern Windows Package Manager ecosystem. It focuses exclusively on online software management, allowing you to search, install, and upgrade thousands of applications directly from Microsoft's official repositories. It ensures you always have the latest, most secure versions of your software, all handled through a clean and automated command-line interface.

Download Script

Designed for speed and convenience in offline environments. This simple yet powerful drag-and-drop utility takes a folder of your pre-downloaded `.exe` and `.msi` installers and attempts to run them all silently and sequentially. It's the perfect tool for setting up a new machine without an internet connection or for installing specific software versions.

A comprehensive system hygiene tool for meticulously uninstalling applications and removing all their traces. It goes beyond the standard uninstaller by deep-scanning for leftover files (which it safely moves to a quarantine folder) and orphaned registry keys. With its multi-layered confirmation system for registry deletion, it combines power with safety.
